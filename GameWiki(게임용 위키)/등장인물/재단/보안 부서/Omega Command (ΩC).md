# 1. 개요
***
MTF로 부족한 대규모 격리 실패 상황이나 케테르등급이상의 SCP가 탈출했을때 파견된다.
이 부대가 출동한다는것은 해당 기지에 생존자는 없을 확률이 아주 높다.
